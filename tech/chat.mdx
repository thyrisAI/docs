# <span style={{ color: '#50E3C2' }}>`GET /api/chat`</span>

The `/api/chat` endpoint allows you to send requests for both **text-to-text** and **text-image-to-text** models.

---

## **Endpoint Details**
- **Method:** <span style={{ color: '#50E3C2' }}>`GET`</span>  
- **Authorization:** `Bearer <your_api_key>`  
- **Content-Type:** `application/json`
