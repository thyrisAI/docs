# <span style={{ color: '#6CA0DC' }}>`POST /api/image`</span>

The `/api/image` endpoint allows you to send requests for generating high quality images.

---

## **Endpoint Details**
- **Method:** <span style={{ color: '#6CA0DC' }}>`POST`</span>  
- **Authorization:** `Bearer <your_api_key>`  
- **Content-Type:** `application/json`
